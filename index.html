Counselor Notepad

Counselor Notepad was developed for school counselors who would like to keep all student notes in one location. Counselor Notepad is ideal for:

  • Classroom Observations
  • IEP Meetings
  • 504 Meetings
  • Behavioral Plan Collaborations
  • 1:1 Student Sessions
  • Faculty Meetings

In Counselor Notepad, there is a counselor dashboard which provides an index for users to view a summary of all student notes. In addition, notes can be viewed, updated and deleted in the counselor dashboard. Sign up and try it today!
